** Usage

#+BEGIN_SRC elisp
(require 'dired-details-s)
#+END_SRC

** Key

- "(" : Rotate details combination

** Customize

#+BEGIN_SRC elisp
(setq dired-details-s-types
  '((size-time  . (size time))
    (all        . (perms links user group size time))
    (no-details . ())))
#+END_SRC
